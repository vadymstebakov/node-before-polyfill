# node-before-polyfill
[![npm](https://img.shields.io/npm/v/node-before-polyfill.svg)](https://www.npmjs.com/package/node-before-polyfill)
![npm bundle size (minified)](https://img.shields.io/bundlephobia/min/node-before-polyfill.svg)

A minimal polyfill for [`Node.before()`](https://developer.mozilla.org/en-US/docs/Web/API/ChildNode/before), using MDN's implementation. Mostly meant for IE, Safari, Safari Mobile.

## Install
```sh
$ npm install --save node-before-polyfill
```

## Usage
Pick your favorite:
```js
require('node-before-polyfill');
```
or
```js
import 'node-before-polyfill';
```
